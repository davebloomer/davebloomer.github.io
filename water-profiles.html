Visualising Water Profiles

Posted on Sun 11 April 2021 in python

Here's the first post of a few planned within a fermentation series. One of the key ingredients to any drink, whether beer, kombucha, hop water etc is of course water, and not all waters are created equal. If water is the base of a drink, some flavours are better suited to a blank canvas, while others are complimented by minerality to make certain flavours pop. A common analogy is when cooking, flavours are brought out using salt, vinegar and other condoments. Getting this balance right can make all difference. And conversely if you're starting point is too rich in these properties, it's probably worth considering diluting, or even switching out for a neurtral bottled water. A common recommendation is to look in to the properties of your water at home, investigate what the properties of your water are, and see where you can go from there to tweak your water profile using common additives such as gypsum, baking soda and salt.

The subject can be confusing if you don't have a background in chemistry. Having studied rock chemistry, in theory I should have a head start, but my brain isn't really wired to think in ions. For this reason, I wanted to explore the potential to display this data as a visualisation exercise. Taking a dataset of common source and target water profiles from popular brewing software [1][2]. In general, city water profiles are most suitable with the style of beer for which they are famous, eg. Dublin and Stout (Guinness). These data are supplimented by my own water profile from a water report in Barcelona, and some common mineral waters for reference. The data are reduced to three dimensions to allow plotting using the properties of water hardness (minerality: required for clarity, flavour and stability), alkalinity (low: minimal complexity, too high: lose flavour) and Sulfide/Chloride ratio (low: showcase malt, high: dry, crisp, showcase bitterness). Unfortunately, we ran out of axis at this point, so Sodium (Na) couldn't be included on the plot, but this is probably the most intuitive property: in generally, we don't want too much salt in our drink.

import pandas as pd
import numpy as np

import matplotlib.pyplot as plt

# load data to pandas DataFrame, remove subheading and reset index
data = pd.read_csv(r'data\water_profiles.csv', header=0)
data =  data.drop(index=[0]).reset_index(drop=True)

# set relevant data to type: float
for c in ['Ca2+','Mg2+','Na+','Cl-','SO42-','HCO3-']:
    data[c] = pd.to_numeric(data[c])

# custom functions to derive water properties
def calc_hardness(ca2, mg2):
    return (ca2 / 0.401) + (mg2 / 0.243)

def calc_alkalinity(hco3):
    return hco3 / 1.22

def calc_so4_cl(so4, cl):
    if cl < 0.0001:
        return 0
    else:
        return so4 / cl

# apply functions to DataFrame
data['Hardness'] = data.apply(lambda x: calc_hardness(x['Ca2+'], x['Mg2+']), axis=1)
data['SO4:Chloride'] = data.apply(lambda x: calc_so4_cl(x['SO42-'], x['Cl-']), axis=1)
data['Alkalinity'] = data['HCO3-'].apply(calc_alkalinity)

# scatter plot
profile_type = {'Target':'green',
                'Mineral Water':'blue',
                'Source':'red'}

fig, ax = plt.subplots(figsize=(8,8))
for i, l in enumerate(data['Name']):
    x = data['Hardness'][i]
    y = data['Alkalinity'][i]
    z = (data['SO4:Chloride'][i] + 0.1) * 200
    c = profile_type[data['Type'][i]]
    plt.scatter(x, y, z, c, alpha=0.8, label=l)
    plt.text(x+0.3, y+0.3, l, fontsize=9)

ax.set_xlabel('Hardness')
ax.set_ylabel('Alkalinity')
plt.show()

The graphic brought out a few observations I hadn't really been aware of:

  • I wasn't convinced that my water profile was great, but Barcelona cities water is in good company, sharing a water profile with Edinburgh, and looks suitable without modification for Ambers, Ales and Light Hoppy beers.
  • Unfortunately my alkalinity is too high for most of my favourite styles including Pale Ale and IPA beers, so modification would be required here.
  • Burton (upon Trent) is famous for it's high minerality, but Dortmund possesses even higher minerality.
  • Mineral water falls outside the alkalinty range of common water profiles (anything above 280 generally isn't recommended), so swapping tap for mineral water would commonly make things even worse.

fig = plt.figure(figsize=(10,10))

Adding additives, or a combination of additives, allow us to increase the hardness, alkalinity or tweek the Sulfate/Chloride ratio. For the Barcelona water profile, this isn't often going to be helpful when brewing beer or hop water, as the water already possesses strong hardness and alkalinity. However, using additives to create mineral water profiles remains is an option, eg. adding Calcium Chloride to make San Pelligrino. For styles where soft water is required such as lagers, dilution or using using bottled reverse osmosis water would be necessary, with water additives then used to make any desired water profile.

I enjoyed putting together these graphics and learned a lot about the subject. There's plenty of scope to continue this project through building an interative widgit to allow others to graphically understand there own water profile, so I might come back to it in the future.
